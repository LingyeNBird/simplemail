# Changelog

本文件为项目接手后的功能演进草稿，按时间倒序整理。分界点为 `2026-03-31` 起由 `lingye` 接手后的提交。

## 2026-04-20

### Added
- 新增邮箱续期能力：为已过期邮箱提供后端续期接口。
- 新增邮箱页面续期交互：过期邮箱卡片可直接点击“续期”，并在弹窗中选择续期时长。

## 2026-04-15

### Added
- 新增留置邮件存储模型与测试，用于承接未命中收件邮箱的来信。
- 新增管理员留置邮件接口，支持后台查询留置邮件。
- 新增管理员留置邮件页面，支持在 Web 后台查看留置邮件内容。

### Changed
- 更新 README，使其与当前架构和 API 能力保持一致。

## 2026-04-05

### Added
- 新增多主机名能力：将 `smtp_hostname` 从系统级设置迁移到域名维度。
- 新增域名管理页主机名列表，用于展示系统内所有去重后的 hostname。
- 新增域名状态筛选标签样式，提升状态筛选可读性。
- 新增 Cloudflare 创建时指定 Zone 的能力，支持更灵活的 DNS 自动化。

### Changed
- 优化域名管理页面布局，主机名列表与域名列表并排展示。
- 优化域名管理表格列宽与筛选标签样式，减少换行和视觉抖动。

## 2026-04-04

### Added
- 新增域名列表页与管理页的状态统计。
- 新增正则过滤能力，可按模式筛选域名。
- 新增状态筛选能力，支持按 active / pending / disabled 过滤。

### Fixed
- 修复删除域名时的外键约束问题，删除域名时可级联删除关联邮箱和邮件。
- 将 Cloudflare 自动创建与域名提交验证按钮限制为管理员可见。

## 2026-04-03

### Added
- 新增 Cloudflare MX 记录删除能力。
- 新增批量域名管理 API。
- 新增域名管理页批量操作，并支持与 Cloudflare DNS 删除联动。

### Fixed
- 修复 Cloudflare Zone 查找、多级子域名创建、已有 MX 记录复用与激活逻辑。
- 修复 Cloudflare 查询与编译相关问题，提升域名自动化流程稳定性。

## 2026-04-02

### Added
- 新增 Cloudflare API 自动创建子域名 MX 解析功能。
- 新增域名列表页的 Cloudflare 自动创建按钮与创建弹窗。
- 新增系统设置修改 `serverIP/hostname` 后自动回写 `.env` 的能力。

### Changed
- 强化运行时配置同步，后台轮询会及时使用最新的 `serverIP/hostname`。
- 隐藏系统设置页中的 Cloudflare Token 明文显示。

## 2026-03-31

### Changed
- 将存储架构从 PostgreSQL + Redis 重构为 SQLite + 内存限流，降低部署复杂度。
- 合并为单容器部署方案：统一承载 Go API、前端与 Postfix。
- 调整为 Nginx 提供静态前端并反向代理 API，Go API 仅监听 `127.0.0.1:8081`。

### Fixed
- 修复邮件投递 API 默认端口错误导致的投递失败问题。
- 清理遗留文件，修复内部路由安全限制与 SPA 缓存控制问题。
